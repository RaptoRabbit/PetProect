# PetCalc

Простое веб-приложение калькулятора, созданное на основе туториала с FreeCodeCamp.

## Описание

Этот калькулятор поддерживает:
- Базовые арифметические операции: сложение (+), вычитание (-), умножение (*), деление (/)
- Десятичные числа
- Обработку edge-кейсов (несколько операторов подряд, повторное нажатие = и т.д.)
- CSP-безопасный код (без использования eval)

## Технологии

- HTML5
- CSS3 (Grid Layout)
- JavaScript (ES6+)

## Запуск

1. Клонируйте репозиторий или скачайте файлы.
2. Откройте файл `index.html` в любом современном браузере.

Для удобной разработки с автоматической перезагрузкой установите расширение "Live Server" в VS Code и запустите его для папки проекта.

## Структура проекта

- `index.html` - Основная HTML-страница
- `styles.css` - Стили калькулятора
- `script.js` - Логика калькулятора
